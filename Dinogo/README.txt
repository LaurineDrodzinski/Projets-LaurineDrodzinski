
## Présentation de Dinogo

L'élève aura pour but de construire le plus de dinosaures possibles, pour cela il devra bien répondre aux questions, plus il enchaîne les bonnes réponses plus le dinosaures se construit rapidement. 

S'il répond rapidement il gagne encore plus de pièces pour la construction !

Comment jouer ?	
Joueur 1 : avec les touches : W X C
					
Joueur 2 : aves les touches : 1 2 3
					
Pour Quitter le jeu : touche : 'q'

## Utilisation de Dinogo



./compile.sh            // lancer la compilation des fichiers
                        


./run.sh dinogo     (dans un terminal)
